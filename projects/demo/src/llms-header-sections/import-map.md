## Common Import Mistakes

### ❌ Wrong:

```typescript
import {TuiButton} from '@taiga-ui/kit'; // Wrong package!
import {TuiCalendar} from '@taiga-ui/core'; // Wrong package!
import {TuiDay} from '@taiga-ui/core'; // Wrong package!
```

### ✅ Correct:

```typescript
import {TuiButton} from '@taiga-ui/core'; // Core component
import {TuiCalendar} from '@taiga-ui/kit'; // Kit component
import {TuiDay} from '@taiga-ui/cdk'; // CDK type
```
