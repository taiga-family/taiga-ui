TuiKeypad is an on-screen grid primitive for touch input. It only lays out and styles the keys — you populate it with your own tuiKeypadButton elements and wire clicks, labels and disabled state yourself, so the pad can be anything: a numeric pad, a shuffled PIN terminal, a calculator or a custom chooser.

Put tuiKeypadButton on a native <button> or <a> — it stays a real element, so (click) , aria-label , disabled and href all work as usual. Pair it with the tuiKeypadInput directive to drive a field without the native mobile keyboard.
