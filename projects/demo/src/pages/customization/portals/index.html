You can easily create your custom portals by extending our abstract classes and put your own portal-host on any layer
@for (example of examples; track example) { @switch ($index) { @case (1) { Portals can also be used to render content in a separate browser window } @case (2) { Portals can also be used to render content in picture in picture mode } } }

  1. Create your own portal service by extending TuiPortalService

  2. Create your own portal host by extending AbstractTuiPortalHost

  3. Put the created portal host on a desired layer
