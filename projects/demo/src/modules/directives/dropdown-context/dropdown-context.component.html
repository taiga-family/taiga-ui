Tips:

  1. Use Arrow Up/Down to focus closest element.
  2. To close dropdown:
    • Use Esc
    • Make mouse left click outside of dropdown content
    • Open new dropdown via mouse right click
    • Declare local template variable which access to property close of template context object and call it
      (for example, <ng-template let-close="close"></ng-template> ).
  3. To get ActiveZone of opened dropdown you should declare local template variable which access to property activeZone of template context object
    (for example, <ng-template let-activeZone="activeZone"></ng-template> ).
