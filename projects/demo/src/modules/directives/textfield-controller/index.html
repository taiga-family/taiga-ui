Directive allows to customize TuiPrimitiveTextfield with parameters that it adds into DI tree.

That means that you can use the directive on any root element and it will customize all textfields in this scope. Most Taiga UI form controls are based on TuiPrimitiveTextfield and thus accept these options.

If there are several controllers above textfield, their values will be merged. The nearest directives are in priority.
