Компонент InputCount предназначен для ввода целых чисел. Не рекомендуется использовать для значений больше 999.
Позволяет задать минимальное и максимальное значение. При попытке ввода числа, не попадающего в заданный диапазон, значение автоматически заменится на подходящее.
Индикация валидации не предусмотрена. Количество счетов Компонент формы disabled (через formControl.disable()) Скрыть кнопки Минимальное значение Максимальное значение (значение по умолчанию — Infinity) Подпись после числа Шаг изменения значения кнопками
