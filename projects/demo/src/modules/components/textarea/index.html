Textarea uses Textfield to create a multi-line string input. Component would grow from minimal amount of rows until given maximum number of rows, after which scroll will be introduced. You can pass limit to enable counter and extra characters highlight. This would also add validator. If you want to hard limit the field you can use native maxLength attribute. It is possible to override default behavior to introduce your own highlight or other cosmetics. Being built on top of Textfield , Textarea supports most of the same configurations, such as icons on both sides. Minimum number of rows in height Maximum number of rows before scroll appears
