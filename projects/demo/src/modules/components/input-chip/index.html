InputChip = Textfield multi + Chip + ❤️

Textfield[multi] is used to display array of items. By default they are presented as plain strings. Use *tuiItem directive to provide custom representation. You can use tui-input-chip out of the box or implement your own. The context is TuiContext<{ item: T, index: number }> You can customize many aspects of the component, from standard textfield options like icons and tooltips to changing the appearance of each individual chip. Individual tags can be disabled using disabledItemHandler , this would prevent them from being edited, removed or added. Note: keep in mind cleaner would still empty the control. If you want a different behavior, you can disable built-in cleaner and provide your own. InputChip can act as multiple value input with suggestions or as multi-select. With disabledItemHandler you can force the component to only accept suggested values. Component can be used with Maskito to facilitate input masking. Component can be used together with RTL direction. Ability to enter unique or non-unique tags String or RegExp to separate tags
