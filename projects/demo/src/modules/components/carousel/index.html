Carousel allows you to rotate through arbitrary items. Multiple items can be shown simultaneously **— and, with the new `[loop]` input, the sequence can now rotate _infinitely_** (when enabled the last slide flows back into the first and vice-versa).
{{ item }}
Enables seamless infinite scrolling.
• true — the list wraps around, so “next” after the last slide shows the first one again.
• false — stops at both ends (default). Whether or not the slider can be dragged by clicking and holding. Duration in milliseconds for automatic rotation (use 0 to disable). Number of slides shown at the same time. Current slide index (0-based, refers to the original list even in loop mode).

Horizontal padding for every slide.
