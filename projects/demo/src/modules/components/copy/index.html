The tui-copy component provides an elegant way to copy text content to the clipboard. It displays the content normally and shows a copy button on hover, with visual feedback when content is copied.

Key Features

  • Hover Interaction: Copy button appears on hover with smooth opacity transitions
  • Visual Feedback: Shows "Скопировано" (Copied) hint when content is successfully copied
  • Accessibility: Supports keyboard navigation and screen readers
  • Flexible Content: Accepts any HTML content via ng-content
  • Automatic Cleanup: Hint disappears after 2 seconds automatically

Examples Overview

  • Basic: Simple text content with copy functionality
  • Single Line: Longer text content with custom styling (ellipsis for overflow)
