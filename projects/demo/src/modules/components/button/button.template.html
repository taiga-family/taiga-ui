TuiButtonModule — модуль с компонентами tuiButton, tuiIconButton для использования с нативными button и a. Ссылки не могут иметь disabled состояние. Для создания неактивной ссылки нужно убрать href атрибут, а в случае routerLink передать пустую строку. Также, чтобы пустой routerLink не фокусировался с клавиатуры нужно повесить на неё директиву [tuiFocusable]="false".
