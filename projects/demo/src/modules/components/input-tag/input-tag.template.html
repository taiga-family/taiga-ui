InputTag представляет собой поле для ввода тегов с возможностью автокомплита с вариантами в выпадающем списке.
Выпадающий список по умолчанию имеет ограничение по высоте в 400px, которое можно изменить.
Для вставки в DOM выпадающего списка используется Portal, он должен присутствовать на странице. См. раздел о вставке вспомогательных компонентов.
