This component is being refactored. Soon (next major release) you will see the fresh version of it!

Of course, we keep backward compatibility in mind (for 2.x.x). You can still use old version of InputSlider .

However, if you are going to use this component, we recommend to use new version. To enable the "new version"-mode, add new directive. Example:  <tui‑input‑slider new ...>
