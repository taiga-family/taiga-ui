Scrollbar реализует кастомный скроллбар, соответствующий общей стилистике Taiga UI.
Нативный скроллбар просто прячется, что позволяет полностью сохранить специфичное для платформы поведение прокрутки. Получить скроллящийся контейнер можно через токен TUI_SCROLL_REF. Это может быть необходимо для работы с библиотеками для виртуального скролла, которым нужно передать контейнер.

Передать элемент для скролла можно с помощью директивы tuiScrollable. В данном примере используется виртуальный скролл из @angular/cdk.
