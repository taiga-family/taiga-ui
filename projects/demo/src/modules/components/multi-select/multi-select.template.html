MultiSelect представляет собой поле с выпадающим списком для выбора нескольких из предложенных вариантов.
Выпадающий список по умолчанию имеет ограничение по высоте в 400px, которое можно изменить.
Для вставки в DOM выпадающего списка используется Portal, он должен присутствовать на странице. См. раздел о вставке вспомогательных компонентов. Выберите счёт списания
{{item}}

Компонент формы disabled (через formControl.disable()) Значение нативного поля для подписки на ввод и передачи снаружи (сбрасывается в null при выборе элемента) Хэндлер отключенности отдельных пунктов выпадающего списка Использовать поле ввода Высота поля увеличивается, чтобы уместить все тэги Функция определения равенства объектов (нужна в случае, если выбранное значение задано изначально и не присутствует в items, однако там есть точная копия, которую можно определить по какому-нибудь критерию.
