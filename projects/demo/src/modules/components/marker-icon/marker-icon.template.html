Компонент MarkerIcon предназначен для вставки изображений в формате SVG в круглой рамке. Цвет задаётся через параметр mode. При необходимости его можно задать стилями на компоненте.

Маркерные иконки должны иметь размер 48х48px или менее. Иконки меньшего размера будут центрироваться. При размере s они будут показаны в натуральную величину, в остальных размерах они будут масштабироваться соответствующим образом. Модификация Размер Код SVG-изображения или имя иконки
