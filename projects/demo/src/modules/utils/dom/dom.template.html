Набор функций для упрощения взаимодействия с DOM деревом
tuiCustomEvent
Генерирует кастомное событие
getActualTarget
Возвращает target события из Shadow DOM, если событие произошло внутри него. Иначе вернет target обычного DOM.
getClipboardDataText
Возвращает текст из data в clipboardEvent. Работает также в браузерах IE и Edge.
getClosestElement
Возвращает ближайший родительский элемент, подходящий по указанному селектору. Иначе возвращает null. Аналог element.closest().
getDocumentOrShadowRoot
Вернет shadowRoot, если таковой имеется. Иначе вернет document.
getScrollParent
Возвращает ближайший родительский элемент, в котором присутствует скролл.
isElementAtPoint
Проверяет, попадает ли переданная точка в заданный элемент
isElementObscured
Проверяет, перекрыт ли элемент другими или проскроллен вне зоны видимости. Делает это с запасом в 2 пикселя на округления.
