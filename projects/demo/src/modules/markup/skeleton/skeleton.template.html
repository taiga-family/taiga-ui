Скелетоны задаются классом .tui-skeleton. При навешивании этого класса на элемент поверху накладывается фигура, повторяющая форму элемента.

Также существует набор модификаторов, которые надо использовать в паре с основным классом:

  • .tui-skeleton:
    • .tui-skeleton_light: вид для темного заднего фона
    • .tui-skeleton_rounded: повторять скругление
    • .tui-skeleton_short: короткий формат
    • .tui-skeleton_center: короткий отцентрированный
    • .tui-skeleton_text: модификатор для текстовых контейнеров
Классы входят в импорт @taiga-ui/core/styles/taiga-ui-angular-global и доступны глобально. Импортировать их в компонент не надо.
