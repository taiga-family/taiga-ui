Инструкция

Taiga UI использует CSS custom properties для задания цветов и стилей компонентов. Для работы в браузерах без их поддержки присутствует полифилл. Чтобы сменить тему нужно переопределить переменные для цветов в :root блоке CSS а для более сложных кейсов переопределить стили внутреннего компонента tui-wrapper. Всё это можно делать в реальном времени через ThemeSwitcher.
