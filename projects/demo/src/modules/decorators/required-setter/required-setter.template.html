Декоратор для инпутов с сеттером.

См. консоль

quantity min

Данный декоратор презназначен для недопуска undefined значений на вход компонента. При получении undefined сеттер для инпута вызван не будет. При этом в DevMode ангуляра в консоли появится сообщение о том, что это скорее всего случилось по ошибке, так как это несовпадение ожидаемого типа и переданного значения.

Опционально можно передать на вход функцию-контракт вида (input: T[K], context: T) => boolean, где вторым аргументом будет инстанс компонента, в котором происходит проверка. При неуспешном прохождении проверки об этом также будет выведено сообщение в консоль в DevMode. Дополнительно можно предать произвольное количество аргументов для дополнения стандартного сообщения.

Опциональный аргумент — функция-контракт.

Внимание: Angular Compiler не позволяет стрелочные функции в декораторах при включённом strictMetadataEmit (по умолчанию). Чтобы использовать стрелочные функции, необходимо добавить комментарий прямо над @Component вашего компонента
